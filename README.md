This repository contains personal skills.

To use one, drop the skill directory into `~/.codex/skills`, `~/.claude/skills`, or `~/.agents/skills`.

## Skills

- [`wwdc-session-notes`](./wwdc-session-notes/) - Fetch Apple Developer WWDC session transcripts, Code tab snippets, and About tab resources, then write them to Markdown.
